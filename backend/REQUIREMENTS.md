# 后端待实现需求列表

当前 `backend` 已提供静态资源托管、材料上传、异步任务、SSE 事件流、Legato CLI 必需调用和轻量导出。生产能力仍需补齐以下接口与服务。

## P0

1. 材料上传与对象存储
   - 简历必传，成绩单和其他材料可选。
   - 返回文件 ID、文件类型、解析状态、错误码。
   - 支持文件大小限制、类型校验、病毒扫描、删除和保留周期。

2. 简历结构化解析 API
   - 将 `Agents/legato` 简历 workflow 暴露为 HTTP 服务。
   - 输出基础信息、教育背景、证书奖项、经历、置信度和证据来源。
   - 支持 PDF、DOCX、Markdown、图片 OCR 回退。

3. 成绩单解析与课程能力映射
   - 输出课程、学期、成绩、GPA、学分和课程类别。
   - 将课程映射到岗位能力维度。
   - 扫描版成绩单需要稳定 OCR 服务。

4. 岗位能力模型与 JD 解析
   - 维护岗位族、能力项、权重、达标标准。
   - 根据学生材料自动生成推荐岗位。
   - 后续可扩展为粘贴 JD 的定向分析模式。
   - 输出结构化岗位能力要求。

5. 能力评分与雷达数据引擎
   - 输出四维分值、雷达图数据、TOP5 匹配岗位。
   - 每个分值需要解释、证据引用和置信度。
   - 支持规则评分与模型评分合并。

6. 用户数据安全与权限
   - 学生材料包含隐私数据，需要认证、授权、脱敏日志。
   - 支持审计记录、下载权限和数据删除。

## P1

1. 成长路径规划生成
   - 输出阶段目标、周任务、资源链接、达标标准。
   - 根据岗位权重和学生短板调整任务优先级。
   - 支持任务完成状态和再规划。

2. 结构化导出服务
   - 能力画像导出 JSON 和 Excel。
   - 路径规划导出 PDF 和 Word。
   - 匹配结果导出可视化报表。

3. 异步任务与 SSE 事件契约
   - 当前已有内存态任务和可恢复事件流。
   - 生产环境需要持久化 run 状态、标准错误码、超时、重试和多实例恢复策略。

## P2

1. 岗位推荐数据源
   - 接入岗位库或招聘平台数据。
   - 按地区、学历、技能和意向过滤。
   - 推荐理由需要可解释。

2. 可观测性与高可用
   - 指标、日志、追踪、限流、熔断。
   - 无状态 API 服务，持久化任务与文件元数据。
   - 多实例部署下保证任务状态可恢复。
